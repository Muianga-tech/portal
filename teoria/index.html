Módulo 1: Arquitetura e Manutenção de Computadores

Neste módulo, vais compreender como funciona o ecossistema físico de um computador e como diagnosticar problemas.

1.1 O Ciclo de Processamento de Dados

Qualquer computador opera com base em três etapas fundamentais: Entrada (via teclado, rato), Processamento (efetuado pela CPU e apoiado pela Memória RAM) e Saída (exibido no monitor ou impressora).

1.2 Componentes Críticos

  • Motherboard (Placa-Mãe): Onde todos os componentes se conectam elatricamente.
  • Processador (CPU): Onde ocorrem os cálculos aritméticos e lógicos. Mede-se em Gigahertz (GHz).
  • Armazenamento (SSD vs HDD): Os HDDs utilizam discos mecânicos lentos; os SSDs usam chips de memória flash eletrónica, sendo até 10 vezes mais rápidos para iniciar o sistema.

Módulo 2: Fundamentos de Redes & Internet

Aprende como os computadores conversam entre si e como a informação viaja pelo mundo.

2.1 O que é uma Rede de Computadores?

É um conjunto de dispositivos interconectados que partilham recursos, dados e serviços (como impressoras ou acesso à Internet).

2.2 Tipos de Redes Geográficas

  • LAN (Local Area Network): Rede local de curto alcance, usada em salas de aula, escritórios ou residências.
  • WAN (Wide Area Network): Rede de longa distância que interliga cidades ou países. A Internet é a maior WAN do planeta.

2.3 Equipamentos Essenciais

O Switch conecta dispositivos dentro da mesma rede local, enquanto o Router (Roteador) conecta redes diferentes e encaminha os dados para a Internet.

Módulo 3: Produtividade com o Pacote Microsoft Office

Domina as ferramentas indispensáveis para o mercado corporativo e administrativo.

3.1 Microsoft Word (Processador de Texto)

Focado na criação de documentos profissionais, relatórios, cartas formais e currículos baseados nas normas académicas de formatação de texto.

3.2 Microsoft Excel (Folhas de Cálculo)

Ferramenta matemática avançada para gestão de orçamentos e relatórios financeiros. Essencial dominar:

  • Fórmulas Básicas: SUM (Soma), AVERAGE (Média), COUNT (Contagem).
  • Funções Lógicas: A função IF (SE) para tomadas de decisão automatizadas nas células.

Módulo 4: Lógica de Programação e Algoritmos

Desenvolve o teu raciocínio lógico para criar softwares, aplicações e páginas Web.

4.1 O que é um Algoritmo?

Um algoritmo é uma sequência passo a passo de instruções finitas e lógicas para resolver um problema específico. Uma receita de cozinha, por exemplo, é um algoritmo.

4.2 Estruturas de Controle

Na programação, controlamos o fluxo do código usando condições:

SE (nota >= 10) ENTÃO
    Escrever("Aluno Aprovado")
SENÃO
    Escrever("Aluno Reprovado")
FIM_SE
